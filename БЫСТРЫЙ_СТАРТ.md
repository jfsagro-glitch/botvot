# Быстрый старт

## Шаг 1: Настройте ваших ботов

Вы уже создали ваш **Sales Bot** (GameChangerQ_bot). Теперь вам нужно:

### Создайте Course Bot
1. Перейдите в Telegram и напишите `@BotFather`
2. Отправьте `/newbot`
3. Следуйте инструкциям для создания второго бота (например, "GameChangerQ_CourseBot")
4. Скопируйте токен и обновите `COURSE_BOT_TOKEN` в `.env`

### Получите ваш Admin Chat ID
1. Отправьте любое сообщение вашему Sales Bot: `t.me/GameChangerQ_bot`
2. Откройте: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
3. Найдите ваш chat ID в ответе (ищите `"chat":{"id":123456789}`)
4. Обновите `ADMIN_CHAT_ID` в `.env`

### Создайте Telegram группы (Опционально, но рекомендуется)
1. Создайте Telegram группу для общего обсуждения
2. Создайте другую группу для премиум пользователей (опционально)
3. Добавьте обоих ботов как администраторов в группы
4. Отправьте сообщение в каждой группе
5. Снова проверьте URL getUpdates, чтобы найти ID групп (они отрицательные числа, например `-1001234567890`)
6. Обновите `GENERAL_GROUP_ID` и `PREMIUM_GROUP_ID` в `.env`

## Шаг 2: Установите зависимости

```bash
pip install -r requirements.txt
```

## Шаг 3: Инициализируйте базу данных и уроки

```bash
python scripts/init_lessons.py
```

**Важно**: Отредактируйте `scripts/init_lessons.py`, чтобы добавить ваш реальный 30-дневный контент курса перед запуском!

## Шаг 4: Протестируйте ваш Sales Bot

```bash
python -m bots.sales_bot
```

Затем:
1. Откройте Telegram и перейдите к `t.me/GameChangerQ_bot`
2. Отправьте `/start`
3. Вы должны увидеть информацию о курсе и варианты тарифов
4. Выберите тариф и протестируйте оплату (мок-оплата автоматически завершается через 5 секунд)

## Шаг 5: Запустите обоих ботов

Вам нужно запустить обоих ботов одновременно:

**Терминал 1 - Sales Bot:**
```bash
python -m bots.sales_bot
```

**Терминал 2 - Course Bot:**
```bash
python -m bots.course_bot
```

## Напоминание о безопасности

⚠️ **ВАЖНО**: Ваш токен бота чувствителен! 
- Никогда не коммитьте `.env` в git (он уже в `.gitignore`)
- Не делитесь вашим токеном публично
- Если токен скомпрометирован, отзовите его через @BotFather и создайте новый

## Следующие шаги

1. **Добавьте контент курса**: Отредактируйте `scripts/init_lessons.py` с вашим 30-дневным курсом
2. **Настройте сообщения**: Обновите сообщения ботов в `bots/sales_bot.py` и `bots/course_bot.py`
3. **Протестируйте процесс оплаты**: Завершите тестовую покупку от начала до конца
4. **Протестируйте доставку уроков**: Убедитесь, что уроки отправляются автоматически
5. **Протестируйте обратную связь по заданиям**: Отправьте задание и протестируйте обратную связь администратора
