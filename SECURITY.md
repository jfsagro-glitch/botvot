# Безопасность

## ⚠️ ВАЖНО: Токены были скомпрометированы

Если вы видите этот файл, значит токены ботов попали в публичный репозиторий.

### Что нужно сделать СРОЧНО:

1. **Отзовите старые токены**:
   - Перейдите к @BotFather в Telegram
   - Используйте команду `/revoke` для обоих ботов
   - Создайте новые токены

2. **Обновите .env файл**:
   - Замените все старые токены на новые
   - Убедитесь, что `.env` в `.gitignore`

3. **Проверьте активность ботов**:
   - Проверьте логи ботов на подозрительную активность
   - При необходимости заблокируйте ботов временно

## Защита секретов

✅ **Защищено в .gitignore**:
- `.env` - файл с токенами
- `data/` - база данных
- `*.db`, `*.sqlite` - файлы БД
- `*.log` - логи

✅ **Используйте .env.example**:
- Шаблон без реальных данных
- Безопасно для публикации

## Рекомендации

1. Никогда не коммитьте `.env` файл
2. Используйте переменные окружения
3. Регулярно ротируйте токены
4. Используйте GitHub Secrets для CI/CD
5. Проверяйте код перед коммитом на наличие секретов
