# Инструкция по настройке бота

Ваш токен для Sales Bot уже получен. Вот как его настроить:

## Шаг 1: Создайте файл .env

Создайте файл `.env` в корне проекта со следующим содержимым:

```env
# Токены Telegram ботов
# Токен Sales Bot (StartNowQ_bot)
SALES_BOT_TOKEN=your_sales_bot_token_here

# Токен Course Bot - СОЗДАЙТЕ ВТОРОГО БОТА ЧЕРЕЗ @BotFather
COURSE_BOT_TOKEN=ваш_токен_курс_бота_здесь

# ID чата администратора (для обратной связи по заданиям)
ADMIN_CHAT_ID=ваш_admin_chat_id_здесь

# ID Telegram групп
GENERAL_GROUP_ID=ваш_general_group_id_здесь
PREMIUM_GROUP_ID=ваш_premium_group_id_здесь

# База данных
DATABASE_PATH=./data/course_platform.db

# Настройки курса
COURSE_DURATION_DAYS=30
LESSON_INTERVAL_HOURS=24
```

## Шаг 2: Получите ваш Admin Chat ID

1. Откройте Telegram и перейдите к: `t.me/StartNowQ_bot`
2. Отправьте любое сообщение боту (например, `/start`)
3. Откройте в браузере этот URL:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
4. Найдите ваш chat ID в JSON ответе. Он будет выглядеть так:
   ```json
   "chat":{"id":123456789,"first_name":"Ваше Имя"}
   ```
5. Скопируйте это число и обновите `ADMIN_CHAT_ID` в `.env`

## Шаг 3: Создайте Course Bot

Вам нужен второй бот для доставки курса:

1. Перейдите в Telegram и напишите `@BotFather`
2. Отправьте `/newbot`
3. Следуйте инструкциям для создания бота (например, назовите его "GameChangerQ Course Bot")
4. Скопируйте полученный токен
5. Обновите `COURSE_BOT_TOKEN` в `.env`

## Шаг 4: (Опционально) Настройте Telegram группы

1. Создайте Telegram группу для общего обсуждения
2. Создайте другую группу для премиум пользователей (опционально)
3. Добавьте обоих ботов как администраторов в группы
4. Отправьте сообщение в каждой группе
5. Снова откройте URL getUpdates, чтобы найти ID групп (они отрицательные числа, например `-1001234567890`)
6. Обновите `GENERAL_GROUP_ID` и `PREMIUM_GROUP_ID` в `.env`

## Шаг 5: Протестируйте вашего бота

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Инициализируйте базу данных:
   ```bash
   python scripts/init_lessons.py
   ```

3. Запустите Sales Bot:
   ```bash
   python -m bots.sales_bot
   ```

4. Протестируйте в Telegram:
   - Перейдите к `t.me/StartNowQ_bot`
   - Отправьте `/start`
   - Вы должны увидеть информацию о курсе и варианты тарифов

## Важное напоминание о безопасности

⚠️ **Храните ваш токен в безопасности!**
- Никогда не коммитьте `.env` в git (он уже в `.gitignore`)
- Не делитесь вашим токеном публично
- Если ваш токен скомпрометирован, отзовите его через @BotFather и создайте новый
